# 便签记事本
这是一个简单的便签类记事本，可以添加任务、行程、规划、目标等，拥有编辑、添加、删除等功能
是一个特别适合新手的第一个vue2项目，也是一个用来巩固自己知识的好项目
帮助你快速了解vue的基础知识
如果您有更好的方法，也请您指点一二！

## Project setup
```
npm install
```
